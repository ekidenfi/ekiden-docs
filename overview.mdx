# Overview

The Ekiden WebSocket API provides real-time access to market data and account events through persistent, low-latency connections.

Use the public endpoint for market data and the private endpoint for authenticated account events, including orders, executions, positions, and balances.

For message formats, payload schemas, and channel definitions, see the **WebSocket Reference**.

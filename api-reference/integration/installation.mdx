Install and configure the Ekiden SDK for TypeScript. This section walks through prerequisites, installation, and your first API call. Python and Rust SDKs will be added later.

### **Prerequisites**

- **Ekiden account** with production credentials.
- **Node.js** v14 or higher.
- **Package Manager**: npm, yarn, pnpm, or bun.
- **System clock synchronized** (e.g., via NTP) â€“ authentication will fail if your clock is off.

### **Installation**

```bash
# pnpm
pnpm add @ekiden/ts-sdk

# npm
npm install @ekiden/ts-sdk

# yarn
yarn add @ekiden/ts-sdk

# bun
bun add @ekiden/ts-sdk
```



## Configuration

- **REST (Gateway)**: https://api.production.ekiden.fi/api/v1
- **SDK WebSocket**: wss://api.production.ekiden.fi/ws
- **GraphQL HTTP**: https://hasura.production.ekiden.fi/v1/graphql
- **GraphQL WS**: wss://hasura.production.ekiden.fi/v1/graphql

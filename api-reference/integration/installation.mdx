title: "Installation"
---

Install the Ekiden TypeScript SDK using your preferred package manager.

## Install

### npm

```bash
npm install @ekidenfi/ts-sdk
```

### pnpm

```bash
pnpm add @ekidenfi/ts-sdk
```

### yarn

```bash
yarn add @ekidenfi/ts-sdk
```

### bun

```bash
bun add @ekidenfi/ts-sdk
```

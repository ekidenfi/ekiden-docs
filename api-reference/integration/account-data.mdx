### **REST**

- **Root Account**: `GET /api/v1/user/root-account`
- **Sub Accounts**: `GET /api/v1/user/sub-accounts`
- **Positions**: `GET /api/v1/position/list?sub_account_address=<address>`
- **Executions (Fills)**: `GET /api/v1/execution/list?sub_account_address=<address>`
- **Account Balances (Vaults)**: `GET /api/v1/account/balance`
- **Statistics**: `GET /api/v1/account/statistics`

Timestamps:

- REST objects include `timestamp_ms` (milliseconds). The legacy `timestamp` (seconds) remains temporarily for backward compatibility.
